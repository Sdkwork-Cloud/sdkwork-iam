# apps/sdkwork-iam-common/packages

Cross-architecture IAM TypeScript packages shared by every client surface (`sdkwork-iam-pc`, `sdkwork-iam-h5`, `sdkwork-iam-flutter-mobile`, and downstream product applications).

Contracts, SDK ports, service facades, runtime bootstrap, application bootstrap, and RPC contracts live here. UI packages for a specific architecture belong under the matching application root in `apps/`.

Owner: `sdkwork-iam` maintainers.
