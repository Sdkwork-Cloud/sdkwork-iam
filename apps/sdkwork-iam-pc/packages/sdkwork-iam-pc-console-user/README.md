# @sdkwork/iam-pc-console-user

Tenant-owner profile and password self-service for the IAM PC user console. Uses `@sdkwork/iam-app-sdk` through `@sdkwork/iam-service` only.

## Exports

- `createSdkworkIamConsoleUserController`
- `SdkworkIamConsoleUserWorkspace`
- `IAM_PC_CONSOLE_USER_ROUTES`
