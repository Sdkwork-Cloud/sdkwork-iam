# @sdkwork/iam-pc-admin-user

Backend-admin user directory management for IAM operators. All mutations flow through `@sdkwork/iam-backend-sdk` via `@sdkwork/iam-service`.

## Exports

- `createSdkworkIamUserAdminController`
- `SdkworkIamUserAdminWorkspace`
- `IAM_PC_ADMIN_USER_ROUTES`

## Route

`/admin/iam/users`
